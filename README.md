# Clements 49

## Goal

Design a 49-string double-action concert pedal harp, voiced for church solo hymn playing and jazz, with a body of unidirectional + woven flax fabric, resin-infused (VARTM). Start from the TurboSquid model as the base geometry. Build the frame from flax resin-infused molds, and produce the brass parts by lost-wax / lost-PLA casting. Use multiple agents and CPUs working in parallel to achieve this goal.

## Specification

- **Strings:** 49 strings (0G to 7A; concert-tension nylon, gut, and wires)
- **Weight:** 96.5 lbs
- **Dimensions:** 75.5″ H × 40.5″ ExW
